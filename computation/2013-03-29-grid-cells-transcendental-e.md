# The Sense of Place: Grid Cells in the Brain and the Transcendental Number e

**arXiv:** [1304.0031](https://arxiv.org/abs/1304.0031)  
**Published:** 2013-03-29  
**Categories:** q-bio.NC

**Authors:** Xue-Xin Wei, Jason Prentice, Vijay Balasubramanian

**Abstract excerpt:** Grid cells respond when an animal occupies a periodic lattice of "grid fields." The paper proposes that the grid system minimizes the number of neurons required to encode location using a scale that involves the transcendental number *e*.

**Relevance to symbolic/math representation:** Directly links neural coding (grid cells) to geometric/vector representations and efficient encoding of space — a core example of how the brain handles geometry and numbers via vector-like codes.

**Connection to accumulation:** Shows how structured spatial memory (a form of "symbolic" representation) is implemented with minimal neurons.
