# A Computational Model of Systems Memory Consolidation and Reconsolidation

**arXiv:** [1703.01357](https://arxiv.org/abs/1703.01357)  
**Published:** 2017-03-03 (v7: 2019-09-27)  
**Categories:** q-bio.NC

**Authors:** Peter Helfer, Thomas R. Shultz

**Abstract excerpt:** In the mammalian brain, newly acquired memories depend on the hippocampus for maintenance and recall, but over time the neocortex takes over these functions (systems memory consolidation). The paper presents a computational model of this transformation and reconsolidation.

**Relevance:** Excellent for understanding long-term memory accumulation and how memories become distributed/cortex-independent over time. Directly models the "accumulation" process.

**Verified mechanisms:** Provides a testable computational framework for consolidation.
